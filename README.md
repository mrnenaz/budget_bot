# 💰 Budget Bot

Семейный бот для учёта бюджета. Node.js + TypeScript + MongoDB (FerretDB).

## Возможности

| Команда | Описание |
|---|---|
| `/expense` | ➖ Добавить расход |
| `/income` | ➕ Добавить доход |
| `/report` | 📊 Отчёты (сводный, по категориям, топ трат) |
| `/history` | 📋 Последние 10 транзакций |
| `/categories` | 📂 Просмотр и управление категориями |
| `/cancel` | ❌ Отменить действие |

📎 **PDF выписка** — просто отправьте файл боту, он автоматически распарсит транзакции.

---

## Отчёты

- 📊 **Сводный** — доходы, расходы, баланс за период
- 📂 **По категориям** — распределение расходов с прогресс-барами
- 🏆 **Топ трат** — 10 самых крупных расходов

Каждый отчёт доступен за **неделю**, **месяц** или **год**.

---

## PDF парсер

Универсальный парсер работает с выписками большинства российских банков:
- Сбербанк, Тинькофф, ВТБ, Альфа-Банк, Газпромбанк, Райффайзен и другие

⚠️ Работает только с **текстовыми** PDF (не со сканами).

После парсинга показывается превью — количество и суммы транзакций. Категории назначаются автоматически по ключевым словам.

---

## Категории по умолчанию

🛒 Продукты · 🚗 Транспорт · 🏠 ЖКХ · 🏥 Здоровье · 🎉 Развлечения · 👗 Одежда · 🍽 Рестораны · 💊 Аптека · 📱 Связь · 📦 Прочее

Пользователи могут добавлять свои категории. Удалять пользовательские категории может только администратор.

## Быстрый старт

```bash
npm install
cp .env.example .env
nano .env
npm run build
pm2 start dist/index.js --name budget-bot
pm2 save
```

### .env

```
BOT_TOKEN=           # Токен от @BotFather
MONGODB_URI=mongodb://localhost:27017/budget_bot
ADMIN_TELEGRAM_ID=   # Ваш Telegram ID
```
